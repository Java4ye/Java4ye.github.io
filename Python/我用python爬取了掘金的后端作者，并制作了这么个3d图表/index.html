# Http2.0

Http2.0 有几个新特点：

  • 新的二进制格式
  • 多路复用（MultiPlexing）
  • header 压缩
  • 服务端推送（server push）

Http2.0 中头部压缩的原理：(谷歌技术专家大佬分享的图片)

简单来说呢，就是客户端和服务端都会维护一个静态表格，然后在传输信息时，就可以直接传递这个 header 对应的索引给到对方，不用像之前那样传输 header:get ，可以减少传输大小
