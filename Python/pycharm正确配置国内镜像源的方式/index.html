# 国内镜像源推荐

  1. 阿里 https://mirrors.aliyun.com/pypi/simple/
  2. 清华 https://pypi.tuna.tsinghua.edu.cn/simple/
  3. 豆瓣 https://pypi.douban.com/simple/

注意这里都是 https 的，不然下载包的时候会出现其他错误的，如下面的 bs4。

# 在 pycharm 中添加国内镜像源，步骤如下

  1. Ctrl+Alt+s 打开设置，点击第一个红框，再点击第二个红框
  2. 在弹出框中点击 Manage Repositories 即第三个红框
  3. 在新的弹出框中点击加号（第四个红框）添加国内镜像源即可

# pycharm 虚拟环境（venv）中安装 bs4

python3 安装 bs4 报错:
ERROR: Could not find a version that satisfies the requirement bs4 (from versions: none)
错误信息如下图：

根据提示信息，他推荐我们在 venv 虚拟环境中执行这个 pip install --index-url http://mirrors.aliyun.com/pypi/simple/ bs4 🐷
